## Overview

Clearlooks-Phenix-Dark is a fork of the venerable Clearlooks-Phénix theme by Jean-Philippe Fleury (https://github.com/jpfleury/clearlooks-phenix).

This port is developed on Linux Mint 17.3 Mate (GTK2 and GTK3) desktop environment. This environment is the primary target, however patches and pull requests are welcome for other environments.

This is currently a work in progress.
